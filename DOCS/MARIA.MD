#### Maria

Using this type, you can `restart` or `stop` or `start` Maria db:

#### Reset

```yaml
- name: "Maria Service"
  type: "maria"
  command: "restart"
```
#### Stop

```yaml
- name: "Maria Service"
  type: "maria"
  command: "stop"
```
#### Start

```yaml
- name: "Maria Service"
  type: "maria"
  command: "start"
```
